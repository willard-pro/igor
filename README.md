# Igor
Bash project which provides common functionality used by developers.  

# Usage

```
Usage: example-cli [options] [command]

Options:
  -h, --help                 output usage information

Commands:
  create <name>              Create a new project with the specified name
  delete <name>              Delete the project with the specified name
  list                       List all available projects
  info <name>                Display detailed information about a specific project

Examples:
  # Create a new project named "my-project"
  $ example-cli create my-project

  # Delete the project named "old-project"
  $ example-cli delete old-project

  # List all available projects
  $ example-cli list

  # Display detailed information about the project named "existing-project"
  $ example-cli info existing-project
```
